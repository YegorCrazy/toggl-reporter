Приветствуем всех новоприбывших пользователей! Эта программа считывает данные о заданиях пользователей Toggl по их API-Token и переводит данные об их активности в Google-таблицу.

Код написан на Python 3.7, для его использование необходим интерпретатор.

Необходимые библиотеки:
- requests
- urllib
- simplejson
- google-api-python-client
- oauth2client
- httplib2

Каждая из библиотек устанавливается командой:
pip3 install <название_библиотеки>
в командной строке.

Также для работы программы вам понадобится API-ключ Google-sheets. Процесс его получения и настройки API хорошо описан в статье https://habr.com/ru/post/483302/ (читать до установки библиотек).

В этом репозитории, кроме кода, есть файл config.json - файл настроек. Значения пунктов в нем можно менять (после двоеточия и до запятой, в кавычках). Значения пунктов:

TimeZoneUTC - ваш временной пояс в формате [+,-] и число.
CredentialsFile - имя файла-ключа Google (см. пред. пункт)
SpreadSheetID - ID вашей Google-таблицы. По умолчанию пустое поле, после начала работы ID автоматически появится.
StartDay, StartMonth, StartYear - день, месяц и год начала отслеживания действий сотрудников
EndDay, EndMonth, EndYear - день, месяц и год конца отслеживания действий сотрудников
ВНИМАНИЕ! Поля дня и месяца заполняются в формате двух цифр (т.е. не "9", а "09")
Workers - массив с данными о работниках, его можно расширять копированием элементов.
Name - имя сотрудника, используется только для вашего удобства (хоть горшком назовите)
APICode - код, который можно найти в настройках учетной записи работника.

Для корректной работы в одной директории должны лежать файлы code.py, commands.py, config.json и файл-ключ Google. Для запуска программы откройте файл code.py.
